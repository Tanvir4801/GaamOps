## FIX 1 — NEW USER REGISTRATION FLOW

Currently app goes login → OTP → directly to profile setup.
This is confusing. Fix to match Rapido-style flow:

### New Customer Registration Flow:
Welcome Screen
→ "Customer તરીકે ચાલુ" button
→ LoginScreen(role: 'customer')
→ Phone number entry
→ OTP verify
→ Check Firestore users/{uid}
→ EXISTS → go to CustomerMainShell (returning user)
→ NOT EXISTS → CustomerRegistrationScreen (new user)
Fields:
- Full Name (required)
- Village selection dropdown (9 villages)
- Profile photo (optional, skip button)
Submit → save to users/{uid} → CustomerMainShell



### New Saathi Registration Flow:
Welcome Screen
→ "Gaam Saathi બનો" button
→ LoginScreen(role: 'saathi')
→ Phone number entry
→ OTP verify
→ Check Firestore users/{uid}
→ EXISTS with role 'saathi' → SaathiMainShell
→ EXISTS with role 'customer' → show role conflict dialog
→ NOT EXISTS → SaathiRegistrationScreen (new saathi)
Page 1 — Personal Info:
- Full Name (required)
- Village dropdown (required)
- Profile photo (optional)
Page 2 — Vehicle Info:
- Vehicle type selector (Bike / Auto / Cycle)
shown as 3 big tappable cards with icons
- Vehicle number (GJ-XX-XX-XXXX format)
- Vehicle color (optional text)
Page 3 — Confirmation:
- Summary of all info
- "સાથી બનો / Become Saathi" button
- Terms: "GaamRide સેવાની શરતો સ્વીકારું છું"
- Submit → save to users + saathis → SaathiMainShell



Create these files:
- lib/screens/auth/customer_registration_screen.dart
- lib/screens/auth/saathi_registration_screen.dart
  (3-page PageView with progress indicator)

```dart
  Widget _vehicleTypeCard(String type, String emoji,
      String en, String gu) {
    final selected = _vehicleType == type;

  Widget _summaryRow(String label, String value) {
    return Padding(
      padding: EdgeInsets.symmetric(vertical: 6),
      child: Row(children: [
        Text('$label:', style: TextStyle(fontSize: 13,
          color: Colors.grey, fontWeight: FontWeight.w500)),
        SizedBox(width: 12),
        Expanded(child: Text(value, style: TextStyle(
          fontSize: 14, fontWeight: FontWeight.bold))),
      ]),

  Future<void> _submitRegistration() async {
    setState(() => _isLoading = true);
    final uid = FirebaseAuth.instance.currentUser!.uid;
    final phone = FirebaseAuth.instance.currentUser!.phoneNumber ?? '';

    String photoUrl = '';
    if (_profilePhoto != null) {
      // Upload to Firebase Storage
      final ref = FirebaseStorage.instance
        .ref('profiles/$uid.jpg');
      await ref.putFile(_profilePhoto!);
      photoUrl = await ref.getDownloadURL();
    }

    final batch = FirebaseFirestore.instance.batch();

    // users collection
    batch.set(
      FirebaseFirestore.instance.collection('users').doc(uid),
      {
        'uid': uid,
        'name': _nameCtrl.text.trim(),
        'displayName': _nameCtrl.text.trim(),
        'phone': phone,
        'role': 'saathi',
        'village': _selectedVillage,
        'profilePhoto': photoUrl,
        'fcmToken': '',
        'isBlocked': false,
        'createdAt': FieldValue.serverTimestamp(),
        'updatedAt': FieldValue.serverTimestamp(),
      },
    );

    // saathis collection
    batch.set(
      FirebaseFirestore.instance.collection('saathis').doc(uid),
      {
        'uid': uid,
        'name': _nameCtrl.text.trim(),
        'phone': phone,
        'village': _selectedVillage,
        'vehicleType': _vehicleType,
        'vehicleNumber': _vehicleNumCtrl.text.trim().toUpperCase(),
        'vehicleColor': _vehicleColorCtrl.text.trim(),
        'isAvailable': false,
        'isOnline': false,
        'isBlocked': false,
        'isVerified': false,
        'rating': 5.0,
        'totalRides': 0,
        'fcmToken': '',
        'position': {
          'geohash': '',
          'geopoint': GeoPoint(0, 0),
        },
        'lastSeen': FieldValue.serverTimestamp(),
        'createdAt': FieldValue.serverTimestamp(),
      },
    );

    await batch.commit();

    if (!mounted) return;
    Navigator.pushAndRemoveUntil(context,
      MaterialPageRoute(builder: (_) => const SaathiMainShell()),
      (route) => false);
  }
}
```

```jsx
                  ⭐ {saathi.rating?.toFixed(1) || '5.0'}
                </td>
                <td className="text-center text-sm font-medium">
                  {saathi.totalRides || 0}
                </td>
                <td className="text-center">
                  {saathi.isVerified
                    ? <span className="text-green-600 text-xs">✅ Yes</span>
                    : <button
                        onClick={() => verifySaathi(saathi.id, saathi.name)}
                        className="text-xs bg-orange-100 text-orange-700
                          px-2 py-1 rounded-lg hover:bg-orange-200">
                        Verify
                      </button>}
                </td>
                <td className="px-4 py-3">
                  <div className="flex items-center gap-2">
                    {/* Edit */}
                    <button
                      onClick={() => {
                        setSelectedSaathi(saathi);
                        setShowEditModal(true);
                      }}
                      className="p-1.5 text-blue-600 hover:bg-blue-50
                        rounded-lg transition"
                      title="Edit">
                      ✏️
                    </button>
                    {/* Block/Unblock */}
                    <button
                      onClick={() => toggleBlock(saathi)}
                      className={`p-1.5 rounded-lg transition
                        ${saathi.isBlocked
                          ? 'text-green-600 hover:bg-green-50'
                          : 'text-orange-600 hover:bg-orange-50'}`}
                      title={saathi.isBlocked ? 'Unblock' : 'Block'}>
                      {saathi.isBlocked ? '🔓' : '🔒'}
                    </button>
                    {/* WhatsApp */}
                    
                      href={`https://wa.me/${saathi.phone?.replace('+', '')}`}
                      target="_blank"
                      className="p-1.5 text-green-600 hover:bg-green-50
                        rounded-lg transition"
                      title="WhatsApp">
                      💬
                    </a>
                    {/* Delete */}
                    <button
                      onClick={() => deleteSaathi(saathi)}
                      className="p-1.5 text-red-500 hover:bg-red-50
                        rounded-lg transition"
                      title="Delete">
                      🗑️
                    </button>
                  </div>
                </td>
              </tr>
            ))}
          </tbody>
        </table>
      </div>

      {/* Add Saathi Modal */}
      {showAddModal && (
        <AddSaathiModal
          onClose={() => setShowAddModal(false)}
          onSave={addSaathi}
          villages={VILLAGES}/>
      )}

      {/* Edit Saathi Modal */}
      {showEditModal && selectedSaathi && (
        <EditSaathiModal
          saathi={selectedSaathi}
          onClose={() => setShowEditModal(false)}
          onSave={(updates) => updateSaathi(selectedSaathi.id, updates)}
          villages={VILLAGES}/>
      )}
    </div>
  );
};
```
