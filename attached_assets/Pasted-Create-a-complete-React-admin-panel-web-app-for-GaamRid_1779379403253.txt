Create a complete React admin panel web app for GaamRide — a rural ride-hailing and vehicle-booking app in Gujarat, India.

STACK (do not deviate):
- React 18 + Vite
- Firebase v9 modular SDK (Firestore + Auth — NO Firebase Admin SDK)
- React Router v6 (createBrowserRouter)
- TailwindCSS v3 (CDN via index.html is fine)
- react-hot-toast for all success/error messages
- No Redux, no Zustand, no other UI library

=== src/hooks/useCollection.js ===
Real-time Firestore hook using onSnapshot.
Signature: useCollection(collectionName, ...queryConstraints)
Returns: { data: Array, loading: Boolean, error: String|null }
- data items include the doc id as item.id
- unsubscribes on unmount
- catches errors and puts message in error state
- loading starts true, goes false after first snapshot

=== src/hooks/useDoc.js ===
Real-time single document hook.
Signature: useDoc(collectionName, docId)
Returns: { data: Object|null, loading: Boolean, error: String|null }
- Returns null data if doc doesn't exist
- Same unsub + error pattern as useCollection

TIMESTAMP HANDLING (critical — do this everywhere):
  Never call .toDate() directly without checking:
  const formatDate = (ts) => {
    if (!ts) return '—';
    if (ts?.toDate) return ts.toDate().toLocaleString('en-IN');
    if (ts instanceof Date) return ts.toLocaleString('en-IN');
    return '—';
  }

MONEY FORMATTING:
  const formatMoney = (n) => n != null ? '₹' + Number(n).toLocaleString('en-IN') : '—'
