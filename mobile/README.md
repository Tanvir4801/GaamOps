# GaamRide Mobile App

Flutter mobile app for GaamRide ‚Äî rural transport & logistics platform for Mahuva Taluka, Gujarat.

## Setup

1. **Install Flutter**: https://docs.flutter.dev/get-started/install
2. **Firebase setup**: Run `flutterfire configure` to generate `lib/firebase_options.dart`
3. **Google Maps**: Add your Maps API key to `android/local.properties`:
   ```
   GOOGLE_MAPS_API_KEY=your_key_here
   ```
4. **Run**:
   ```bash
   cd mobile
   flutter pub get
   flutter run
   ```

## Key Rules (from spec)
- `status` is always a single string (never pipe-separated)
- Distance from `Geolocator` is in **meters** ‚Üí divide by 1000 for km
- Navigation after login uses `pushAndRemoveUntil`
- Collections: `users`, `saathis`, `rides`, `haul_bookings`, `haul_vehicles`, `villages`, `app_settings/config`
